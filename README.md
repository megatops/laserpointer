# Laser Pointer

![GitHub License](https://img.shields.io/github/license/megatops/laserpointer?logo=license) ![GitHub top language](https://img.shields.io/github/languages/top/megatops/laserpointer) ![GitHub contributors](https://img.shields.io/github/contributors-anon/megatops/laserpointer) ![GitHub forks](https://img.shields.io/github/forks/megatops/laserpointer) ![GitHub Repo stars](https://img.shields.io/github/stars/megatops/laserpointer)

This utility replaces the mouse pointer in Windows 10/11 with a laser dot, making it suitable for use in scenarios such as screen presentations. Run it once to apply the replacement, and run it again to restore the original pointer. The pointer switch typically takes 3–5 seconds to take effect, so please wait a moment.

You can replace the `LaserPointer.cur` to any Windows cursor file to change the laser pointer style.
